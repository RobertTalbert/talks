How many subsets does an $n$-element set have?

(Decomposition) $\emptyset \rightarrow \emptyset$; $\{x_0\} \rightarrow \emptyset, \{x_0\}$; $\{x_0, x_1\} \rightarrow \emptyset, \{x_0\}, \{x_1\}, \{x_0, x_1\}$, ...

(Pattern) It's $2^n$!

(Abstraction) For all $n \in \mathbb{N}$, a set with $n$ elements has $2^n$ subsets.

WHY?

(Solution/Algorithm) The empty set has 1 subset. If a $k$-element set has $2^k$ subsets for some $k$, then a $(k+1)$-element set has $2^k$ subsets with $x_{k+1}$ and $2^k$ subsets without $x_{k+1}$. Hence there are $2^{k+1}$ subsets with $k+1$ elements.
